# CellMigrationLab Docs Hub

Welcome to **docs-hub**, the central documentation repository for **CellMigrationLab**.

This repo is our *single source of truth* for lab-wide documentation that is reused across projects. Project-specific documentation should usually live in the project repository, but should link back here for shared practices and standards.

## Where to find things

### Documentation

All documentation lives under [`docs/`](docs/) and is organized by topic. 

### Images and attachments

- **Images:** [`assets/images/<topic>/...`](assets/images/)
- **Other small files:** [`assets/files/<topic>/...`](assets/files/)
